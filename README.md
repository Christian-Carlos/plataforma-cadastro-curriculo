# Plataforma de Cadastro de Currículos

## Descrição
Sistema web desenvolvido com Google Apps Script para cadastro e gestão de currículos. Permite criação, edição e validação de dados, com integração ao Google Sheets e armazenamento de arquivos no Google Drive.

---

## Funcionalidades
- Cadastro de usuários por CPF
- Edição de dados mediante autenticação por senha
- Validação de campos obrigatórios
- Integração com CEP automático
- Upload de arquivos (foto, currículo, comprovantes)
- Controle de atualização de dados

---

## Tecnologias
- Google Apps Script
- JavaScript
- HTML
- CSS
- Google Sheets
- Google Drive

---

## Estrutura do Projeto

### Backend — Code.gs
Responsável pela validação de dados, autenticação do usuário, integração com a planilha e controle de armazenamento de arquivos.

### Frontend — index.html
Interface do usuário com formulário de cadastro, validações visuais, upload de arquivos e interação com o backend.
